# Finance Tracker

A web application to manage purchase orders (POs) and vendors.

## Features
- Create, update, and delete purchase orders.
- Track active, upcoming, and historical POs.
- Approve POs and move them to the active tab.

## Technologies
- Next.js
- Supabase
- Tailwind CSS
- TypeScript
